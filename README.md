# Todo App

A modern task management application.

## 🎯 Features

- Create, edit, and delete tasks
- Task categories and tags
- Due dates and reminders
- Priority levels
- Progress tracking
- Team collaboration
- Cloud sync
- Dark mode support

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/none-ai/todo-app.git
cd todo-app

# Install dependencies
pip install -r requirements.txt

# Run the application
python app.py
```

Visit http://localhost:5000

## 📋 Requirements

- Python 3.8+
- Flask
- SQLAlchemy
- See `requirements.txt` for full list

## 🛠️ Tech Stack

- Python 3
- Flask
- SQLAlchemy
- JavaScript
- HTML/CSS
